# InterLink
Mobile app for international migrants struggling to find a support system or looking for low pressure ways to socially adapt to their new country

## Core features
- Personality test/cultural onboarding survey to personalize group suggestions 
- Pre-created social groups based on culture, hobbies, geolocation, and more which users can join
- Community social events based on group themes for users to participate in
- Group chat for members to communicate with each other in real time 
- Create custom social events for a various social groups

## Tech Stack
- React Native
- Expo
 
## Requirements
- Expo Go (download from playstore or app store)

## Installation
- git clone https://github.com/dnng1/CS4352Group16.git
- cd frontend
- npm install
- For Expo-specific packages, ensure you're using the correct Expo SDK version (54.0.18) as dependencies are version-locked to ensure compatibility


## Running App
- npx expo start
- Scan the given QR code through the expo go app (if using android) or camera app (if using ios) 

## Limitations
- no backend yet
- lacks database for user info and group details
